📁 How to Add a New Wireframe

  1. Create your HTML wireframe file and save it as index.html
  2. In the GitHub repo, click Add file → Upload files
  3. Type the folder name first, e.g. product-page/index.html — GitHub creates the folder automatically
  4. Commit the file — it goes live at yourusername.github.io/sycomore-wireframes/product-page/
  5. Add a new card to this index.html file so it appears in the hub (copy any existing card below)
